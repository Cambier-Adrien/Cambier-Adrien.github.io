The Android project comes to fruition

As part of a collaborative project, we have developed an Android application designed to visualise Wireshark frames. Our approach involves the generation of a file from Wireshark, followed by its processing by a Java extractor dedicated to translating the data into JSON format. This information is then transmitted to a web page, enabling the Android application to display it efficiently.

This project is of particular value to me, as it has enabled me to familiarise myself with the development of Android applications, a field I'm deeply interested in. It has also considerably strengthened my understanding of the Java language. I'm delighted to share this achievement, which illustrates my ability to tackle complex projects and my ongoing commitment to deepening my technical skills.
