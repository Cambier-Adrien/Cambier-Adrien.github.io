Lan network simulation

The project involved setting up a local network for a small company using Packet Tracer software. The overall topology is made up of 4 sub-sections:
- An internal network for employees
- A dns and dhcp server accessible only to the internal network
- A DMZ server accessible from the outside and the internal network
- An external server for simulating machines outside the LAN

These 4 sub-sections are all connected to a firewall that manages packet traffic according to source. This project has enabled me to familiarize myself with Packet Tracer and the various routing protocols I've seen. Attached is a report detailing the project's stages.
